# Z Token

The Z token is the governance and value-capture token for Z Protocol. Revenue from the entire DeFi stack flows to Z token holders, while also granting governance rights.

Fixed supply: 1 billion Z tokens. Token distribution details are to be determined.

---

## Token Utility

### Gas

The Z token is used to pay transaction fees on Z.

ZEC holders can access all core applications — trading, lending, borrowing, staking — without touching the Z token. When ZEC is used as gas, it is swapped into Z at the protocol level. The user never handles the conversion.

### Staking and Consensus

Z token stakers are one of the three legs of ZSP consensus. By staking Z tokens to validators, holders help secure the network and earn consensus rewards. Z stakers receive the largest share of consensus emissions (2/3 of total).

Staking Z tokens alongside ZEC through [Dual Staking](./05-dual-staking.md) earns boosted rewards.

For how staking works, see [Consensus (ZSP)](./04-consensus-zsp.md). For liquid staking tokens, see [Liquid Staking](./06-liquid-staking.md).

### Governance

Z token holders participate in protocol governance: parameter changes, validator policy, emission schedule adjustments, and other protocol-level decisions.

### Value Accrual

Revenue generated by Zcosystem applications accrues to the Z token. As usage grows across Z Trade, Z Lend, USDZ, yUSDZ, and other Zcosystem primitives, that activity translates into value for Z token holders.

For consensus mechanics, see [Consensus (ZSP)](./04-consensus-zsp.md). For the Zcosystem's applications, see [Zcosystem](./07-zcosystem.md).
